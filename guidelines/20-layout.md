# Layout & Responsive Rules

## Grid

* Desktop: 12 columns, 24px gutters, 64px margins
* Tablet: 8 columns, 20px gutters, 40px margins
* Mobile: 4 columns, 16px gutters, 20px margins
* Large screens: max content width **1280px**

## App shell

* Top bar: logo, search, mic, user/menu
* Center: **Results Canvas** (vertical stack)
* Optional left: History/Saved (collapsible)

## Layout principles

* Content-first: prioritize results
* Breathing room: whitespace improves comprehension
* Clamp long titles, avoid horizontal overflow
* Tables: allow horizontal scroll inside table container only

## Motion

* 150–300ms transitions
* Only animate state changes (recording, streaming, new results)
* No decorative motion
