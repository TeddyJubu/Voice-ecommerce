# Trust Through Transparency (Sources)

## When sources are required

If any factual claim comes from the web, show sources.

## Citation rules

* Inline markers like [1] [2] in summaries where relevant
* Each component can include a Sources section:

  * title
  * domain
  * clickable URL
  * (optional) favicon

## Content rules

* Summarize, don’t copy long passages
* If sources disagree, show both and state the disagreement briefly
* If no reliable source exists: say “Not sure yet” + offer to search
