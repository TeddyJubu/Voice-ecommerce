# Accessibility (WCAG 2.1 AA)

* Contrast: 4.5:1 for normal text
* Target sizes: 44x44px minimum
* Keyboard nav: all interactive elements reachable via Tab
* Visible focus ring: 2px using brand teal
* Semantic structure: h1 → h2 → h3
* ARIA labels for mic/search controls
* Alt text for images

Prefer semantic HTML:

* <button> for actions
* <a> for navigation
* <main>, <nav>, <article>, <aside> landmarks
