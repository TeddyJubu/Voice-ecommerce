# Design Tokens (Perplexity-inspired)

Use tokens (variables), not random values.

## Color tokens

### Primary

* --color-brand-primary: #20B2AA
* --color-brand-accent:  #00C2FF
* --color-action-secondary: #3A86FF

### Neutrals

* --color-bg-light: #FAFAFA
* --color-bg-dark:  #121212
* --color-text-primary: #1D1D1D
* --color-surface-subtle: #F5F5F5

### Accents

* --color-warn:    #F9C74F
* --color-danger:  #FF6F59
* --color-success: #8AC926
* --color-feature: #FB6F92

## Color rules

* **Dark mode first** (bg #121212).
* Text contrast: **4.5:1 minimum**.
* Links + primary actions use **brand teal**.

## Typography

* Font: system-ui, -apple-system, Segoe UI, Roboto, Helvetica, Arial, sans-serif
* Body line-height: **1.5–1.6**
* Line length: **60–80 chars**
* Weights: keep to 3–4 weights max

### Type scale

* Display: 40px / 700
* H1: 30px / 600
* H2: 24px / 600
* H3: 18px / 600
* Body: 15–16px / 400
* Small: 12–13px / 400

## Spacing (8pt rhythm)

* --space-1: 4px
* --space-2: 8px
* --space-3: 16px
* --space-4: 24px
* --space-5: 32px
* --space-6: 48px
* --space-7: 64px
