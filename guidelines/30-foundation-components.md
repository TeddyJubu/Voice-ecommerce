# Foundation Components (Build these first)

## Primitives

* Button (Primary / Secondary / Ghost / Destructive)
* Input
* SearchBar (48–56px height, 24px radius)
* Card (12–16px radius, subtle border)
* Badge (status + citations)
* Skeleton (use sparingly)
* Toast (errors and permission issues)

## Button variants

* Primary: teal fill, white text
* Secondary: transparent + teal border + teal text
* Ghost: no border, teal text, hover only
* Destructive: coral fill, white text

## Card rules

* Padding: 20–24px
* Border: subtle (or none) with very light shadow
* Hover: slight border/shadow change
* Keep surfaces calm: low contrast backgrounds
