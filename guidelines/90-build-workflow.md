# Build Workflow (Velocity over Perfection)

When uncertain, follow this order:

1. Make it work with real components + real data.
2. Make it cognitively fast (hierarchy, spacing).
3. Add sources + refine actions.
4. Only then polish visuals.

## Quality checklist

* Is search/mic the obvious primary action?
* Are results structured components (not chat bubbles)?
* Are sources visible for web-derived facts?
* Are there 1–3 components only on first response?
* Does it work on mobile without breaking?
* Is there a clear next action (Refine / Show more / Open links)?
* Does always-listening mode show pipeline state clearly?
* Is the LLM provider badge visible on results?
* Are session cost metrics accurate and updating?
* Does the intent gate save credits by rejecting ambient speech?
